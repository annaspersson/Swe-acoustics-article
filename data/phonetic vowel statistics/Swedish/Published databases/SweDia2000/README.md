This folder contains txt and csv files with vowel data of two types. 1) Vowel data generated by the Praat vowel analyzer script (/raw-formant-data). 2) Vowel data that has been outlier corrected by the SweDia-2000-outlierCorrection.rmd script (/outlier-corrected-vowel-data). 
The scripts relate to each other in the following way: the outlierCorrection script reads in the raw formant data generated by the Praat script and outputs csv-files that are read in by the main output script that generates the paper.
